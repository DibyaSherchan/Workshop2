Learning about new tags

After building this website I came to a clear understanding of how the semantic tags were supposed to work. I learnt everything from the header, section, footer, article, aside etc were meant to function. Also learnt a few meta tags which I have implemented in this website.

Section, Article, Main Tags

The section tag could be used to divide a specific section of the whole website while using div as a form of CSS initiater. Article tag is used to display the paragraphic section of the website. There is also a similar tag which is used to define the main content of the page, appropriately called main.

Aside Tag

The aside tag can be used to display not so important but important content at a chosen corner of the webpage
Although it needs CSS to show change it is still something that defines a certain part of the webpage
